# Adding Friends

Agora uses a mutual friendship model — both people must accept before they're friends.

## Sending a friend request

1. Go to the person's profile (search for them at **Search → People**).
2. Click **Add friend**.
3. They'll receive a notification. Once they accept, you're friends.

## Accepting a request

- Go to **Friends → Requests** to see pending incoming requests.
- Or click the notification to accept directly from the notification centre.

## Removing a friend

1. Go to their profile.
2. Click **⋯** → **Unfriend**.

---

## Friend Lists

Friend lists let you group friends and control post visibility more precisely.

### Creating a list

1. Go to **Friends → Friend Lists → New list**.
2. Give it a name (e.g. "Close friends", "Work").
3. Add friends to the list.

### Using a list

When composing a post, set the visibility to **Friend List** and choose which list can see it. Only members of that list will see the post.

### My Feeds

You can also build a **Custom Feed** from a friend list — go to **My Feeds** and create a feed with a friend list filter. This lets you see only posts from that group in a dedicated tab.
